Cellfie

A Protégé Desktop plugin for mapping spreadsheets to OWL ontologies. The plugin is available in menu item "Tools > Create axioms from Excel workbook..."

Please refer to the wiki for more details

Version info

v2.1.0 (10-Oct-2016)

  • Implemented check box selection for selecting transformation rules to use when generating the axioms.
  • Implemented easy cell range definition by interactive cell selection in the spreadsheet preview table.
  • Implemented the new MappingMaster feature `mm:EntityIRI` used by the cell reference that will look for a match to an entity name in the target ontology, and extract the IRI if a match is found.
  • Implemented a single log file usage to avoid cluttering the working directory.
  • Implemented key shortcuts for faster editing rules.

Please refer to the release notes for the full description.

v2.0.0 (4-May-2016)

Bug fixes related to generating axioms

Please refer to the release notes for the full description.

v2.0.0-beta-7 (14-Mar-2016)

  • Added new string manipulation function mm:decimalFormat to create a user-specified number format.
  • Added support to rdf:PlainLiteral datatype that allows user to assign a language tag using (xml:lang) directive.
  • Added new options to handle name encoding for generating valid IRI string.
  • It is possible now to specify the annotation value as an IRI by using (mm:IRI) directive.

Please refer to the release notes for the full description.
